## Usage

- Start mysql server with mysql.server start command
- Install database with mysql -u root -p < schema.sql
- npm start to run files

## Requirements

- Node 6.x
- MySQL 5.x


## Development

### Installing Dependencies

From within the root directory:

```sh
 sudo npm install -g mysql
 npm install
```
